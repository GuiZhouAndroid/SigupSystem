# 基于Android的考证报名系统

  1.本项目将近3年前读书时，自学Android时做的，存在功能不完善，代码不规范等问题。
采用多个第三方SDK进行联合开发，由于第三方SDK的更新换代原因，导致项目当前依赖包存在版本兼容性问题，
未解决未维护，所以目前AS新版运行源码存在很多未知原因错误。
因此本源码仅供参考，后续如果我需要进行完善的话，会重新制作。
  2.要看效果就在上面的apk文件，可以在手机上安装。
